# Шаблоны промптов

- **system.txt** — системная инструкция (Chain-of-Thought: шаги → ответ, «Я не знаю» при отсутствии данных).
- **few_shot.txt** — примеры «вопрос — ответ» из той же предметной области для few-shot.
- **main.txt** — основной шаблон. Плейсхолдеры: `{system}`, `{context}`, `{few_shot}`, `{user_query}`.

Изменить тон, правила или примеры можно правкой этих файлов без изменения кода.
