# Задание 2 — база знаний

## Что где лежит

- **`final_fandom/`** — итоговые статьи в формате `.txt` для индексации (основной вход для `task_3/build_index.py`).
- **`terms_map.json`** — словарь замен в формате постановки.
- **`kimetsu_articles/`** — вспомогательные скрипты подготовки корпуса; справочник замен в постановке — **`DICTIONARY.md`** (и связанные заметки, например **`CLEAN.md`**).

## Опционально: скрипты в kimetsu_articles

Из каталога `task_2/kimetsu_articles` (при необходимости правки исходников):

```bash
python3 apply_dictionary.py
python3 clean_edited.py
```
