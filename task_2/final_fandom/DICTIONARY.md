# Словарь замен для обезличивания фандома (сатирическая версия)

**Концепция:** демоны → **алкаши**; боевые термины → **кулинария и варка пельменей**. Организация «истребителей» становится **бригадой варщиков пельменей**, которая «выводит из запоя» (одолевает) алкашей при помощи половников и правильной варки.

## Примечания по использованию

1. **Падежи**: склонять замены (Танджиро → Тимура, демон → алкаша, половник → половником и т.д.).
2. **Единство тона**: везде «алкаши» и «варка пельменей»; «убить демона» → «вывести алкаша из запоя» или «снять с огня» (в зависимости от контекста — буквально про пельмень или иронично про алкаша).
3. **Название произведения**: «Клинок, рассекающий демонов» → «Половник, выводящий из запоя» или «Варщики пельменей против алкашей».
4. **Сатира**: демоны = алкаши; истребители = варщики пельменей; клинки = половники; отрубить голову = снять пробу; глициния = рассол; кровь Мудзана = настойка Козлова; Двенадцать Лун = Дюжина Главных Завсегдатаев.

Если нужно добавить термины или усилить/смягчить сатиру — можно расширить этот файл.
