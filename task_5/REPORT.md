# Отчёт по заданию 5: демонстрация и защита от prompt injection

## Что сделано

- Подготовлен вредоносный документ: `task_5/malicious_doc.txt`.
- Реализована индексация вредоносного документа в отдельную БД:
  - скрипт: `task_5/index_malicious_doc.py`
  - база: `databases/bad/chroma_db`
- Проведена серия тестов с защитой и без защиты.

## Результаты тестов

- В режиме `off` (без защит) зафиксирована утечка (`swordfish`) на провокационных запросах.
- В режиме `protected` утечка блокируется:
  - pre-prompt запрещает выполнять инструкции из документов;
  - post-фильтр отбрасывает подозрительные чанки.
- Лог прогонов сохранён: `task_5/LOG_EXECUTION.md`.

## Ключевой вывод

Защитный слой существенно снижает риск утечки из вредоносных документов при сохранении полезных ответов по нормальным запросам.
